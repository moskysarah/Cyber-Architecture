# Cyber-Architecture
Here we explain every step to learning  in cybersecurity

🛡️ Roadmap Cybersécurité
🏗️ 1. Les fondations indispensables (le "SOC")
🌐 Réseaux
📡 Modèle OSI / TCP-IP en profondeur
🔀 Routage, Switching, VLANs, NAT
📨 Protocoles : ARP, DNS, HTTP/S, TLS, DHCP, Kerberos, SMB
🛠️ Outils : tcpdump, Wireshark, Nmap
💻 Systèmes
🐧 Linux : administration, permissions, processus, systemd, Bash
🪟 Windows : Active Directory, GPO, NTFS, PowerShell, Registre
🖥️ Virtualisation : VMware, VirtualBox, Proxmox
🐍 Programmation / Scripting
🐍 Python (sockets, requests, scapy, impacket)
📜 Bash / PowerShell
⚙️ Langage C (Reverse Engineering & Exploit Development)
🌍 Web
🌐 HTTP/HTTPS, Headers, Cookies, CORS, Same-Origin Policy
🗄️ Bases de données SQL et injections
🛡️ 2. Blue Team (Défense)
📚 À apprendre
📊 SOC / SIEM : Splunk, ELK, Wazuh, Sentinel
📏 Règles de détection : Sigma, KQL, SPL
🛡️ EDR / XDR : CrowdStrike, Defender, SentinelOne
📜 Analyse de logs : Windows Event Log, Syslog, Auditd
🔍 Forensic : Volatility, LiME, Autopsy, Sleuth Kit
🎯 Threat Intelligence : MITRE ATT&CK, YARA, CTI
🚨 IDS / IPS : Snort, Suricata, Honeypots
🚑 DFIR : Incident Response

🏅 Certifications

🎓 Security+

🎓 GCIA
🎓 GNFA
