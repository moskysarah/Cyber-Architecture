<h1>🛡️ Cybersecurity Roadmap</h1>

Here we explain every step to learning  in cybersecurity

<h2>🏗️ 1. Les fondations indispensables</h2>

<h3>🌐 Réseaux</h3>
<ul>
  <li>📡 Modèle OSI / TCP-IP en profondeur</li>
  <li>🔀 Routage, Switching, VLANs, NAT</li>
  <li>📨 Protocoles : ARP, DNS, HTTP/S, TLS, DHCP, Kerberos, SMB</li>
  <li>🛠️ Outils : tcpdump, Wireshark, Nmap</li>
</ul>

<h3>💻 Systèmes</h3>
<ul>
  <li>🐧 Linux : administration, permissions, processus, systemd, Bash</li>
  <li>🪟 Windows : Active Directory, GPO, NTFS, PowerShell, Registre</li>
  <li>🖥️ Virtualisation : VMware, VirtualBox, Proxmox</li>
</ul>

<h3>🐍 Programmation / Scripting</h3>
<ul>
  <li>🐍 Python (Sockets, Requests, Scapy, Impacket)</li>
  <li>📜 Bash / PowerShell</li>
  <li>⚙️ Langage C (Reverse Engineering & Exploit Development)</li>
</ul>

<h3>🌍 Web</h3>
<ul>
  <li>🌐 HTTP/HTTPS, Headers, Cookies, CORS, Same-Origin Policy</li>
  <li>🗄️ Bases de données SQL et injections</li>
</ul>

<hr>

<h2>🛡️ 2. Blue Team (Défense)</h2>

<ul>
  <li>📊 SOC / SIEM : Splunk, ELK, Wazuh, Sentinel</li>
  <li>📏 Détection : Sigma, KQL, SPL</li>
  <li>🛡️ EDR / XDR : CrowdStrike, Defender, SentinelOne</li>
  <li>📜 Analyse de logs : Windows Event Log, Syslog, Auditd</li>
  <li>🔍 Forensic : Volatility, LiME, Autopsy, Sleuth Kit</li>
  <li>🎯 Threat Intelligence : MITRE ATT&CK, YARA, CTI</li>
  <li>🚨 IDS / IPS : Snort, Suricata, Honeypots</li>
  <li>🚑 DFIR : Incident Response</li>
</ul>

<b>🏅 Certifications</b>

<ul>
  <li>🎓 Security+</li>
  <li>🎓 Blue Team Level 1</li>
  <li>🎓 GCIA</li>
  <li>🎓 GNFA</li>
</ul>
